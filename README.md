### RenMLY

MLY (.mly) is a language designed with C++ to create animation videos (rendered with Ren Tool) alongside complex graphical scenes, like fractals in Math.
MLY embraces an easy approach for begginers!

```js
fn sum(x, y) {
  /* implicit return! */
  x + y
}

var a = sum(3, 4)
```

The current state of MLY language or Ren Tool does not reflects the final version since the ongoing version is at alpha of the alpha gamma super alpha gamma alpha beta (like, super new project!)
